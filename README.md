# ExRarible
[![hex.pm version](https://img.shields.io/hexpm/v/ex_rarible.svg?style=flat)](https://hex.pm/packages/ex_rarible)

Rarible API client for Elixir

## Installation

Add the `ex_rarible` package to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_rarible, "~> 0.0.1"}
  ]
end
```
